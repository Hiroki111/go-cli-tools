# go-cli-tools

- col-stats: It reads CSV files and executes `sum` or `avg`. It offers testing with benchmarking.

- notify: *This program is incomplete.* A library that allows you to include system notifications in other applications.

- goci: Continuous Integration tool for Go programs. Check how Go files are organized with sub folders in this program.

- mdp: Markdown preview tool. It convert an MD file to HTML.

- pomo: *This program is incomplete.* Pomodoro tool with interactive interface. 

- pScan: *This program is incomplete.* This tool executes a TCP port scan on a list of hosts similarly to the Nmap command. It allows you to add, list, and delete hosts from the list using the subcommand hosts.

- todo: TODO list that saves data into a specified json file.

- todo-client: Client for the `todo-server` (see below), using REST API.

- todo-server: REST API server for TODO list that uses `todo` program (see above)

- walk: It walks through a directory and prints files inside it.

- word-counter: It counts words, lines or bytes in a file.
